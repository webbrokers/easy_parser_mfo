# Парсер витрин вебмастеров и# Парсер витрин МФО (v1.0)

Система для ежедневного мониторинга позиций офферов с админ-панелью и аналитикой.

## Функции

- **Smart Card Detector (v3.0)**: Умный поиск контейнеров офферов, обход защиты сложных витрин (Sravni.ru и др.).
- **Matching Engine**: Автоматическое объединение дублей компаний ('Zaymer', 'Робот Займер') в единый бренд для точной аналитики.
- **Скриншоты**: Сохранение визуального состояния витрины при каждом запуске.
- **Аналитика**: Расчет средней позиции и популярности брендов по всем витринам сразу.

## Деплой (Railway.app)

1. Сделайте `git push`.
2. Добавьте `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` в переменные (если нужно).
3. Приложение само создаст базу и наполнит её сайтами при первом запуске.

Делаем простенький парсер витрин вебмастеров и мфо.
