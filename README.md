# Парсер витрин вебмастеров и# Парсер витрин МФО (v1.0)

Система для ежедневного мониторинга позиций офферов с админ-панелью и аналитикой.

## Функции

- **Парсинг Puppeteer**: сбор офферов из основной выдачи и попапов (B1, B2).
- **Скриншоты**: автоматическое сохранение скриншота страницы при каждом парсинге.
- **Аналитика**: расчет средней позиции офферов и графики трендов.
- **TG Уведомления**: ежедневный отчет в Telegram (опционально).

## Деплой (Railway.app)

1. Сделайте `git push`.
2. Добавьте `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` в переменные (если нужно).
3. Приложение само создаст базу и наполнит её сайтами при первом запуске.
